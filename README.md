# Setup

1. Download the .exe
2. Run the RealismEngineSetup.exe


# Additional Software

## Package Manager
1. Run npm install RealismEngineBoxer in cmd (Node.js Required!)
2. Find any Package on the Realism Engine website and get the install name
3. Run REPackages add your-wanted-package-install-name
4. go to Plugins > Packages > Installed and find you package
